### numberOfPointers

```ts
numberOfPointers: number;
```

Represents the number of pointers (fingers) currently placed on the screen.

### pointerType

```ts
pointerType: PointerType;
```

```ts
enum PointerType {
  TOUCH,
  STYLUS,
  MOUSE,
  KEY,
  OTHER,
}
```

Indicates the type of pointer device in use.
